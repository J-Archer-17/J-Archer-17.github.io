夜店治疗职业奇穴打法参考

六月 11, 2020 PVE通用

顺序按五七万歌~目前我没打过yx安小逢，所以这里写的老五奇穴都只适用于普通本。

五毒

如果加速不到1729建议第四个奇穴换成冰丝。1729以上可以考虑2060冰丝或者3770虫兽。

老一

除了点名就是奶T。奶毒的减伤太弟弟了，点名一般会有风袖或者春泥笼花，如果保得费力直接千蝶。T脆的话，也可以把千蝶和蝶池都给T，因为没什么群体掉血。老一奶毒就是回蓝工具人。

老二

奇穴没变化，记得驱散T。蛊惑一定不能断，如果两个T恨不得在拉脱的边缘反复横跳，除了蛊惑没人能同时奶两个瓜皮。

反弹一般都是弹远程，所以那会儿跟远程站一起给个蝶池就行了。我发现小号醉舞+2000 +2000 奶不上人，所以洗柔丝的话群加基本靠蝶池和女娲千蝶。

不知道鑫鑫不能奶到底是bug还是机制，如果机制是不能奶，那就必须洗柔丝。冰蚕千蝶可以放心用，蝶池醉舞不能在鑫鑫那边用。

老三

一般的群奶奇穴即可。如果习惯醉舞，第一个奇穴洗清霜。

月落这种分散的选择千蝶，万劫快落地的时候蝶池，毕竟蝶池太高了吃不到。一觞可以蝶池但如果离万劫很近就算了，一觞正常情况不会死人。其他的技能一般不会死人，谨慎一点可以不交大技能，女娲冰蚕就行。

这里是自己的网站终于可以吐槽了（狗头.jpg），从我打本的体验看，小号靠醉舞奶团血很吃力。但是大号就不一样了，别人视频里女娲醉舞就是可以抬起来。何况近几个赛季的本哪有团血压力，AOE技能蝶池千蝶CD完全够用。我遇到比较坑的奶毒技能统计里醉舞太多会吐槽，无脑醉舞啊，但是能奶上的奶毒你撕人家干嘛，就因为别人点的奇穴跟你不一样嘛。

老三可以按宏你敢信？记得下鼎；圣手按几下可以避免划水太明显，被踢了不要说是我的宏。

// 挂机
/cast 迷仙引梦
/cast 冰蚕牵丝     // 点清霜的话这一行换成 /cast醉舞九天

// 全团大掉血
/cast 女娲补天
/cast 蛊虫献祭
/cast 碧蝶引
/cast 千蝶吐瑞
/cast 冰蚕牵丝      // 点清霜的话这一行换成 /cast醉舞九天

老四

英雄生死禁疗不用担心溅射，拉断白链和驱散的时候团血会掉得比较多，所以洗冰体/清霜。

平时除了T没人掉血，AOE技能蝶池千蝶完全够用。T接球的时候掉血严重的话，开个女娲奶，没有女娲就开玄水，别为了不OT直接放生到死。

蛊惑建议给当前T而不是出去排buff的T。因为buff炸一下T不会死，这个T平时是不会被BOSS打的。但是很多属性不太好的T扛老四承伤就很高，当前T才是治疗需要关注的那个。

老五

我觉得枭泣在这里挺好用的，虽然目标有限，但是对于OT的人，盾比掉血再奶更能救急。

为什么说队友是瓜皮就要洗柔丝呢，首先T水的话，你奶到大团那一群OT的DPS了大概率跟着OT。然后那种拉不住仇恨的瓜皮T多数也是不会续减伤的，不拼命奶他他都能给你表演个专业团队。正常的话还是建议洗冰体，下p之后奶毒要奶团血的情况就变多了。

我一般把蛊惑挂给BOSS那边的T，因为奶秀去河边奶湿身队友/奶秀自己在T的时候，BOSS这边就顾不上了。

七秀

老一

除了辞致全部洗单加，奶好T给好点名的减伤就行。辞致其实不是为了机制洗的，而是因为每次都能遇到冲锋路上用脸挡的队友。如果队友机智，辞致可以换成承清。

老二

我觉得承清挂给被叠流血的那个T很好用，这个提高承疗是提高他受到的所有治疗，而不是奶秀一个人的。如果T5层流血没扶摇卡掉也没减伤，就给一个风袖。buff爆炸是穿透伤害，给减伤是为了避免buff爆炸之后马上有平A把T砍死。驱散看好，其他的就没什么了。

普通T压力没那么大，DPS懒得第一时间停手，面向也可能打到人，所以洗出辞致和散余霞在这几个时段抬抬团血。但是其他时候还是只有T掉血，所以双鸾心问不动。

老三

常规群奶奇穴即可，AOE之前多扩散一下翔舞和上元，没什么其他要注意的。个人感觉红隙比较符合这个BOSS的掉血模型。

老四

余寒和红隙根据你习惯选，其他的常规群奶奇穴就行了。

理想情况是，点双鸾奶T，AOE技能开梵音丢王母就行了。然而我之前打的野团，白链/驱散的时候奶妈半天抬不上血，所以有了第二套奇穴= =。

普通本双鸾对于生死符特别舒服，如果担心溅射，只对生符的人按上元-翔舞-翔舞-翔舞…都能避免被锁了。

老五

普通的话这个BOSS能选的方案太多了，因为你不知道队友靠不靠谱。

出小怪之前挂个扶摇，小怪走近的时候二话不说先开个天地。天地的减仇比OT了再减伤好用得多，不用担心天地开早了后面没减伤怎么办，你还可以扶摇跳，风袖没给T就可以给自己。为什么这样讲呢，因为T就算拉了很多小怪，掉血会很快但一般不至于被秒。但是小怪看DPS和奶就容易两刀一个。我发现这里双鸾完全奶得起T，反而是自己和其他的奶/DPS更需要风袖。

奶落水狗开抠jio过去给双鸾首跳就行了，最多再交个王母，debuff吃的治疗量不多，没必要浪费风袖。

关于璇妗，我觉得普通没必要洗，英雄洗了没用。安小逢死人无非是OT/应援按错/下水太远了奶不到。OT是因为T菜/仇恨高的DPS没挂扶摇开减仇减伤；应援按错不应该；下水是因为站错圈/进了粉圈。所以如果到了让奶秀洗5分钟战复的程度，那得死多少人了？遇到这样的队友不是应该切冰心锤爆他们狗头嘛，何必自己给人收拾烂摊子。

万花

老一

实际上老一洗群奶奇穴然后一直提针长针奶T都行，这里的青律就是为了T压力小一点，因为没有什么需要群奶的地方。猛击的时候T没开减伤就给一个，也不用担心被顶，洗微潮就是为了T能用自己更高比例的减伤顶掉你的春泥（狗头）。点名的给春泥或笼花就行；BOSS的连砍笼花会被移除所以奶就是了，要是T倒了肯定是因为没开盾。

老二

青律是为了避免溅射洗的，我觉得老二没有必要洗清风。不推荐让奶花驱散T，因为水月利针太浪费了，读条利针如果预先读，那么T可能后跳出范围驱不到；如果不提前读，那么不如其他治疗的瞬发驱散来得及时。流血T叠到5层debuff如果他没减伤你就给一个春泥，防止debuff爆炸+平A倒T。

老三

无脑刷团的BOSS，比较推荐第二个奇穴。虽然第一个很稳，但是遇到一觞出人群的演员什么的，奶花跑来跑去非常不利于站桩刷血。而且普通治疗溢出太多，述怀还没跳人都满血了。

老四

如果T脆可以洗个青律，这个BOSS也就白链和驱散的时候会炸炸团血，所以不用全程刷持续，没有AOE的时候奶T就行。普通建议青律，避免奶到死符，只是别人奶不奶就不知道了。

老五

奶花老五一般都是奶BOSS T，除非团里两个奶花没必要都盯着BOSS T。如果你奶的是人群这边可以洗个微潮。书离应对减疗很有用，后面减疗叠高了就要多用提针而不是长针来奶T。

关于OT的问题，可以多用扶摇和太阴（春泥重置太阴）。水月要用来保人，如果为了不OT频繁开水月就太浪费了。

长歌

老一

老一破盾的地方不多，可以洗温语增加盾的厚度。这里几个修饰盾的奇穴都是为了保T。因为团血不会掉，就可以解放出洞天来辅助奶T。温语平吟天音能给其他治疗提供一个非常舒适的保T环境，毕竟盾不顶一般的减伤。但是野团不一样，好几次我遇到T承伤高的BOSS，眼看着T残血了，给了盾也半天奶不起来。所以洗犹香，牺牲一点持续减伤（老一没有很疼的那种平A），自己用羽盾来奶T吧。

很多奶歌喜欢用枕流保T，但是我习惯用盾。因为枕流是靠2个GCD生效一次的大量治疗去奶T，保人方面其实并不强。尤其是有队友的情况下，其他治疗职业都是靠数值治疗或百分比减伤，奶歌的盾相当于给T加了一段血条，伤害生效之前就已经有了。如果别的奶妈卡了GCD或者大加晚了一点点，残血的T可能就会死，但是有盾的T就不容易死。

老二

如果可以不洗驱散依旧推荐洞天，但是野团我发现真的没人驱……

T身上流血3~4层时候给盾，切高山读宫。枕流应该会在debuff爆炸的时候生效，读出一次枕流马上切回梅花曲风补盾。这样不是最佳方案但适合懒人。

其他时候就一直补盾补盾补盾，如果乱天狼大半个团都喜欢站进去，反弹也不停手，可以洗寒酒。但我还是觉得温语保T稳一些。

老三

这个BOSS由于频繁补盾，又洗了驱散，会肾亏，所以有奶毒一定要及时嗑鼎。这里不建议洗谪仙，奶歌的盾能保很多人不死，但是放弃寒酒去点谪仙，治疗量上去了，保人就保不住了。如果因为其他治疗不靠谱，外面的团长非要你洗所谓的半盾半血奇穴，那你不如切血歌洗洞天疏曲寸光阴刷HPS，让他看清楚到底是谁演。我打老三化解一般都在1w+，HPS只有几百，团血很稳。

盾歌全程以补盾为主，挂给T的角不要断，影子控制好，因为肾太疼了。奶团的工作交给其他治疗，这里盾的覆盖率越高越不容易死人，而不是你奶得越多越不容易死人。

关于相依要提一句，万劫不要在天上炸相依。有的人是扶摇，有的人是突进BOSS，高度经常不一致，相依可能奶不到人。奶歌可以切高山青霄飞羽用徵奶团，不用担心天上死人。落地的时候如果残血的人还是很多可以炸一个相依，如果很快奶上了就没有必要，把相依留到强化月落问莲什么的。

老四

伤害多数都是穿透伤害，平吟发挥不了什么作用，所以洗犹香。相依给白链，其他的时候还是补盾补盾补盾。

推命阶段吃个影子给T套盾，然后放心奶T，不要T残血了还不敢奶。球刚放完技能可以落一个云生，避免小队成员OT导致减员。

老五

这里犹香盾太香了，OT的人给个盾必不可能死。其他的时候盾就用来保T。

需要奶落水狗的时候用影子赶路，路上给自己挂个角加速，用枕流把这些CXK奶出来。枕流就是洗给落水狗的，并不是为了保T。众所周知奶歌面对减疗很窒息，主T那边更适合用盾来保。人群这边经常破盾，没什么时间对着两个T读枕流，也是犹香盾救人更稳。

兼山需要的时候可以洗，但是只要队友没那么离谱还是建议洗绝唱配合寒酒补盾，老五破盾的地方其实挺多的。
