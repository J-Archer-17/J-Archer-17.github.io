【结庐在江湖】明尊玻璃体光速上手指南

六月 23, 2020 明教

一个粗略的介绍，并不全面，仅供快速上手食用。技能奇穴效果请自行到游戏里查看。

需求秘籍

  • 贪魔体：减CD×3 持续时间×1
  • 极乐引：减CD×3 范围×1
  • 渡厄力：减CD×3 效果×1
  • 贪魔：减CD×3 持续时间×1

装备属性

100级10人本需求105命中，5御。
25人普通夜店需求110命中，5御。
25人英雄夜店需求115命中，8御。

加速阈值只有773和2428。

满足以上条件后，明尊的装备以内外防为主。我觉得生死劫就是外防没有准备多件装备换洗的习惯，所以配装会均衡一点。下面的配装仅供参考，看看属性就好，不用照搬。

奇穴和宏

奇穴仅作推荐，可替换的太多了，每个人习惯不同，你打起来顺手的就是最合适的。

常规

// 常规宏 云端ID：当关关#明尊
/cast 心火叹
/cast 生死劫
/cast [moon>=sun] 银月斩
/cast [sun>moon] 烈日斩
/cast [sun>=moon&sun<100] 赤日轮
/cast [moon>sun&moon<100] 幽月轮

需要自己打戒火。

如果你的加速在1269~2428之间，把【妙境惊寂】换成【辟滞扈沙】。

如果没洗【纵遇善缘】，那么把宏的第二行替换成/cast 净世破魔击。需注意月破带突进。

如果按宏仇恨不稳，自己打几个日破，或者按一会儿下面的群拉宏。

打断在第3个奇穴【寒月耀】。

群拉

群拉奇穴可以根据你的习惯随意组合，多点几个少点几个看需求。

/cast 净世破魔击
/cast 烈日斩
/cast 赤日轮

类似pt夜店老五这种可以把【流沙渡】洗出来。

月斩的仇恨很高，群小怪的时候如果只按宏拉不住，就自己打打月斩。

我原本想洗【净身明礼】，后来发现群斩仇恨值令人迷惑，就放弃了。

打法

我习惯的起手是戒火斩 - 慈悲愿 - 极乐引 - 朝圣言。

先上易伤，保证DPS的爆发能吃到朝圣言+戒火。慈悲愿不是为了强仇而是为了一点点闪避。极乐引一定要在朝圣言之前放避免朝结束OT。我见过的99%起手朝圣言结束OT的喵都是因为没开极乐引。

如果需要留朝就不打，不影响仇恨。

极乐引基本上好了就用，群拉的时候就根据出怪的时间决定。

还见过“我朝圣言接极乐引都能OT，有bug吧”，兄弟我想跟你说95级定军随便接走朝圣言。朝圣言并不附带无威胁气劲，强制是强制，仇恨值是仇恨值，朝圣言期间如果DPS仇恨打得高，朝结束当然会OT了。

二仇在100%~119%之间时，拍个强仇复制仇恨避免OT。

BOSS出点名技能时，被点的队友仇恨超过83%，点名结束要马上拍强仇避免OT。

关于无威胁气劲、OT机制、强仇等级、强仇生效机制等等，这里先不写那么多，可以参考剑三坦克MT职业概览的【仇恨机制】部分。
