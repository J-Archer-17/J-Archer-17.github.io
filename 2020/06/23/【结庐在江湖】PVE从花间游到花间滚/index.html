【结庐在江湖】PVE从花间游到花间滚

六月 23, 2020 花间

花间玩得像CXK，所以来坑害无辜小伙伴了。

需求秘籍

  • 兰摧：2减CD 1减读条 1持续时间
  • 钟林：3减读条 1持续时间
  • 商阳：1持续时间
  • 快雪：3伤害
  • 阳明：2减读条
  • 芙蓉：1效果（刷新dot）

没写的优先读伤害，然后读会心，其他随意。

装备属性

加速阈值

唯一推荐的加速阈值是994，但是校服衣服很尴尬所以会超出一部分，那也不建议再往上堆，影响循环。

命中无双

100级10人普通本需求105命中，15无双。
25人普通夜店需求110命中，30无双。
25人英雄夜店需求115命中，40无双。

请根据团队配置和小吃小药自行调整配装。考虑到二十四桥不一定全程都有，命中小药舍不得吃，建议112+命中。无双随缘，不穿精简怎么都够，穿精简不管无双也没关系（你要是能把无双搞成10当我没说）。

其他属性

我不想算收益了，如果追求DPS可以用dalao做的计算器去算一下。我自己是破防不到55浑身难受，达到55~60再考虑会心。会效靠五彩石，精简我觉得这赛季拍了有点冤大头。

五彩石

推荐会心·破防·会效或破防·攻击·会效。
我是混元的和内功的哪个便宜买哪个……

懒人参考配装

仅供参考，不是属性最合理的配装。因为①我对破防有额外的好感；②毕业装可以避免过高的加速，手感比带衣服的四件套要舒服，对我来说手感比DPS重要（觉得配装问题很大打过我再说）。

如果不是大号毕业装，可以打蓝色附魔，插几个辣鸡石头。

奇穴和输出循环

一般的打法

多数情况下这套奇穴不用动了。

再次强调是懒人打法不是极限打法，不想纠结太多细节，熟悉副本机制比什么都重要。

// 起手
乱洒+阳明 - 商阳 - 玉石
兰摧 - 钟林 - 商阳 - 水月+快雪×6 - 芙蓉 - 快雪 - 玉石

// 常规循环
兰摧 - 钟林 - 商阳 - 快雪×6 - 芙蓉 - 快雪 - 玉石

// 水月CD转好
兰摧 - 钟林 - 商阳 - 水月+快雪×6 - 芙蓉 - 快雪 - 玉石

// 乱洒CD转好
乱洒+商阳 - 阳明 - 快雪 - 玉石


// 水月乱洒CD一起转好
兰摧 - 钟林 - 商阳 - 水月+乱洒+商阳 - 阳明 - 快雪 - 玉石

开水月的时机依实际情况而定，不必照搬。

当你的加速超过1104（穿了校服衣服）时，会发现按照上面的循环打，芙蓉CD还没好dot已经跳完了。因此对循环稍作调整：

// 起手
乱洒+阳明 - 商阳 - 玉石
兰摧 - 钟林 - 商阳 - 水月+快雪直到芙蓉CD转好 - 芙蓉 - 4跳快雪 - 3跳快雪 - 玉石

// 常规循环
兰摧 - 钟林 - 商阳 - 快雪直到芙蓉CD转好 - 芙蓉 - 4跳快雪 - 3跳快雪 - 玉石

// 水月CD转好
兰摧 - 钟林 - 商阳 - 水月+快雪直到芙蓉CD转好 - 芙蓉 - 4跳快雪 - 3跳快雪 - 玉石

// 乱洒CD转好
乱洒+商阳 - 阳明 - 快雪 - 玉石

// 水月乱洒CD一起转好
兰摧 - 钟林 - 商阳 - 水月+乱洒+商阳 - 阳明 - 快雪 - 玉石

如果实战中芙蓉CD没好但是dot马上跳光了，就直接玉石炸掉，不要为了等芙蓉浪费玉石。

特殊情况的打法

需要频繁移动（如战兽山老四）/需要远离BOSS群小怪（如夜店安小逢）的情况下，可以把【兰舟】换成【流离】，挂dot顺序改为兰摧 - 商阳 - 钟林。流离奇穴下玉石CD会转得比较快，基本上好了就爆。这时候芙蓉CD就跟不上玉石了，所以按照玉石CD，一波刷芙蓉一波不刷。

由于【焚玉】的存在，频繁转火一直是花间比较头大的问题，像敖龙岛老二，我当时洗的是【春寒】【流离】。转伞的时候乱洒商阳阳明光速挂dot，然后看队友。如果队友暴力，继续读快雪，伞快要打完的时候芙蓉+玉石；如果队友在打篮球，读一个快雪就炸掉玉石，然后正常挂dot打伞，点了流离玉石CD较短应该能炸到。飞星血厚，没乱洒也可以转。其他的伞就尽量单伞时有乱洒再转，5伞时不转也得转。这个本后面应该没人打了，只是为了解释如何应对频繁转火提一下。

夜店运动姿势推荐

老一小怪受到AOE伤害降低80%，当单体BOSS（精锐，然后周贽）打，你要相信快雪已经长大了，它会自己帮你群小怪了。

BOSS突围时吸收所有伤害但dot的debuff可以挂，在突围结束前挂好dot，突围结束瞬间芙蓉快雪玉石，然后乱洒商阳阳明快雪玉石，之后正常打。

老二站桩撸，反弹打另一个的时候炸完玉石再回去打，总之单体BOSS不要在没挂dot的情况下读快雪，损失太大。

老三瞎打，出人群尽量用迎风回浪，跑路或上天落地的时候不能读条就打商阳，不要小看商阳首跳。可以留意下时间轴，尽量让走位和芙蓉/玉石赶在一起，就不怎么影响循环了。

老四瞎打。

老五可以洗流离，dot挂给血厚的小怪。如果周围小怪多，玉石CD好了也可以等等再炸，先读快雪群怪。BOSS这边输出没有特别需要注意的地方。
