懒人莫问夜店划水指南

六月 11, 2020 莫问

写在前面

因为我是个懒人，就不会有太麻烦的操作，输出也肯定不那么极限，所以这里的内容只适合像我一样的PV邑咸鱼略作参考。

这赛季主要还是大徵和切剑两种输出方案，纯鹿鸣挂按键太划水了，尤其群攻多的时候，大概率被看出来DPS和装分不符，所以不考虑。

大徵需要手动，按宏很难控制，搞不好就输出特别低。大徵莫问不用像近战那样一直劈里啪啦按键盘，其实并不麻烦，所以比较推荐。切剑的话，建议剑技能手动，也很划水，会比一键宏打的高很多。

大徵的打法

大徵流需求装备加速1269+。在没有精简堆会效的情况下，依然建议会心至少35往上，接近毕业的装备可以轻松达到40+。不过掉落看脸，加速不到1269影响比较大可以先打切剑；如果超过1269很多也没关系，再高都不至于没法打。

如果会心低，第一个奇穴改成【音朗】。

循环

首先，切到阳春白雪曲风。我的阳春是挂按键的，手动的话读条时一直按就行，下面的循环里就不写了。

1. 孤影 — (影子+羽)×3 — 收孤影  // 如果懒得开孤影，只打3羽1徵也行

2. 羽 — 商 — 角 — 羽×2 — 徵

3. 宫 — 羽×3 — 徵
   宫×2 — 徵

4. 宫 — 羽×3 — 徵
   宫 — 羽×3 — 徵
   宫×2 — 徵

然后4无限循环。孤影好了可以孤影放一波影子。

解释

大徵流一切都是为了打出更多、更暴力的7跳徵，具体每个奇穴什么效果可以自己看一下，这里就关注几个最重要的，影响循环的奇穴。

这两个奇穴是大徵流打起来很快乐的原因，也决定了大徵不适合按宏，而且需要自己对副本比较熟悉、T和被一些机制点名的队友靠谱，不能拉着BOSS反复横跳。

徵CD的时候就靠宫填充了，这层没什么其他可点的，正好用宫来刷新dot。

书离是大徵流最最核心的一个奇穴，无视防御的效果在越高等级的本作用越明显（所以大战10人我还是喜欢切剑划水而且爆发足够）。

宏打得低除了副本里各种走位会不受控制地断徵、掉dot以外，还有一个原因就是宏不能监控并判断羽充能CD，只能打3羽1徵，这样会浪费徵的CD，打的徵比手动少很多。

注意事项

羽充能问题

宫 — 羽×3 — 徵
宫 — 羽×3 — 徵
宫×2 — 徵

这个循环不一定每次都打得出来，可能有时候羽充能缓不过来了，那就打

宫 — 羽×3 — 徵
宫×2 — 徵

也有时候由于走位、停手等原因，羽充能完全够用，那么可以一直

宫 — 羽×3 — 徵
宫 — 羽×3 — 徵
宫 — 羽×3 — 徵
...

关于补dot

这是我打夜店的技能比例

可见没点洞天疏曲的dot伤害太弟弟了（点了也弟弟！我也想单杀奶妈啊💢），在不需要转火的BOSS占比也只有2%~3%左右。我菜是一方面，另一方面是我觉得打本的时候掉dot一半都是因为处理机制不得不停一下手，这时候徵和羽的CD照样转，与其花2个GCD去补这么弟弟的dot，还不如多打几个徵。按宏的DPS不如手动也有这个原因，宏的判定一般都是dot没了就补，浪费太多本来可以打徵的时间了。

dot除了本身的伤害，还有【师襄】修饰下提高宫伤害的作用。但还是看实战，宫的占比也就那样。所以我习惯单个BOSS如果离战斗结束还有很长时间，就补dot。如果快打完了，就不补，时间用来多打徵和宫。

对于短时间转火或频繁切目标的情况，就可以不挂dot，只打徵羽宫。

适时切曲风

大徵流输出最高的曲风是阳春，但这不代表大徵必须绑定阳春曲风。损失10%的基础攻击并不会真的让你DPS下降10%，基础攻击不是最终输出。

很多人说大徵怕走位，一走位就掉DPS。这是对莫问的一种误解。其实只要不是BOSS被T或点名的人不可预料地带着绕背瞬移，就没什么应对不了的走位情况。

高山曲风使莫问成为唯一一个可全程移动释放技能的远程职业。需要走位就果断切高山，连按两次，开主动高山的加速效果。这个加速十分有用，开了它，你加速二段变三段，三段变四段……想象一下别人扶摇二段小轻功的时候，你上天打出如此密集的7跳徵有多恐怖。

走位的时候大家都掉DPS，只是损失不同而已。只要处理得当，莫问是走位损失最小的职业之一了。

关于转火

详见下面的夜店打法推荐。

关于书离

书离的无视防御时间是有限的。如果因为走位等原因，叠了3层凌冬之后没有马上打徵，那么下一次打徵之前需要看一眼书离还剩多久。如果即将或已经消失，就补一个羽再打徵。

切剑的打法

加速依然推荐1269+。

第二个奇穴需要的时候可以换风入松。

循环

开打前切阳春曲风，孤影放影子，收孤影。

起手切剑，打

剑·羽——3跳剑·宫
剑·羽——3跳剑·宫
剑·羽——3跳剑·宫
...(循环)

注：3跳剑·宫不包括【徊影】修饰的首跳。

剑结束后马上连按两次高山（第一次切曲风，第二次开加速），然后一边放影子一边读变宫。这一部分可以用宏来替代：

/cast 疏影横斜
/cast 高山流水
/cast 变宫

之后清绝影歌CD好了就切阳春，然后切剑，打上面的循环，如此反复。

群攻的时候不打上面的切剑循环，而是直接打剑·宫。

解释

切剑打法唯一需要注意的是清绝影歌需绑定阳春曲风。

这两个赛季新出的奇穴，算是给切剑的一个补偿。

单体BOSS需要打剑·羽—剑·宫的循环，是因为这个奇穴。弄影剑·羽的伤害不低，不过懒得打影响也不大，输出不会低很多。但剑·羽是单攻，所以群的时候没必要这么打。

夜店蹦迪姿势推荐

老一

我一开始切剑打的，后来发现大徵打得更高。因为狼牙刀盾兵会获得buff盾牌武装，受到的aoe伤害降低80%，持续到战斗结束。1 所以群攻对DPS的提升并不多。但是切剑伤害不低，而且省事，所以怎么打看你心情。

大徵

常规奇穴即可。这个BOSS单攻莫问可能需要根据团队群攻的速度转火，比如宫[->精锐] — 羽×3[->盾兵] — 徵[->精锐] …只是举个例子，不是一定这样打。大致的意思就是尽量保持主目标身上的dot，然后其他技能可以切目标打。如果小怪打得快，就没有必要对着小怪读徵以免最后几跳打不出来。如果转火的时候dot掉了，看血量，打得快就不补了。

切剑

砍，砍TMD。

老二

没有群攻，输出环境不算差，推荐大徵。

大徵

开打的时候不挂dot，第一次烟雾弹之后挂dot正常打。留意一下时间轴，反弹之前看羽和徵的CD，要么提前转火，要么反弹之前5秒不读徵。反弹时打另一个，一般打完一个徵就回来继续打。如果因为团队DPS的原因转火次数较多，就不挂dot了。

切剑

砍，砍TMD。

老三

典型单体BOSS，推荐大徵。

大徵

这个BOSS全程需要跑动的时候就切高山跑路，完全不影响输出循环。万劫最省事，直接青霄上天，切高山开加速继续输出。幻歌不想赌脸叠减疗也可以上天。

一觞点名的时候如果队友比较憨，可能没有背对，或没有贴近BOSS，那么BOSS可能往人群突进一截。可以自己提前后退一小步，免得徵读到一半BOSS穿面向，就很伤。

切剑

砍，砍TMD。

老四

水牢能提供的群攻也就那么一两波，所以大徵打得高一些，切剑差距不会很大，看你心情。

大徵

木角还是开高山边跑边打，推命的阶段，等T拉球的时候可以放影子，球到位之后吃个影子打基本不可能OT。队友被锁你能打的技能不多，如果离推命结束还早可以打几个羽。推命快要结束就不要浪费羽了，攒好充能转阶段直接三连7跳徵打出去。

切剑

只要buff不是太阴，切剑CD好了就切。其他没什么要注意的。

老五

我不想跟外面的团翻滚所以只说普通，BOSS这边对DPS需求并不高，推荐切剑。

大徵

像前面老一那里说的，小怪不挂dot，切目标打羽读徵填充宫就行了。打BOSS的时候需要走动就切高山，没什么特别的。

切剑

推荐一套脸皮宛如城墙的奇穴

重点在【长清】【豪情】。长清是群徵，豪情是给徵的首跳加一次伤害。

注意洗了豪情切剑之后奶妈一般就奶不到你了，所以洗这套奇穴要苟好命（也不用太担心，毕竟剑体没血了回自动回本体）。

出小怪切剑砍小怪，如果退出剑体了小怪还没群完，就断徵继续群。这时候阳春还是高山随你，高山开加速读徵很快乐而且方便跑路，阳春基础攻击高。这里莫问不太怕OT，有影子，有青霄飞羽，还有云生。

解释一下这里的“断徵”。因为点出了【豪情】，所以在徵读完第二跳的时候直接读下一个徵，这样第二跳的伤害和下一个徵首跳的伤害是一起打出来的。

原则上切剑CD的时候，3个及以上小怪，打徵。2个就鹿鸣单点血多的吧。

安小逢这边就跟其他的BOSS没什么两样了，有剑切剑，没剑鹿鸣按键开启。
